# FastAPI Inference Server

A FastAPI server for machine learning inference tasks. Provides REST endpoints with automatic Swagger/OpenAPI documentation.

---

## Features

- REST API endpoints for inference
- Async request handling
- Environment variable configuration via `.env`
- Interactive API docs (Swagger UI and ReDoc)

---

## Tech Stack

- Python 3.10+
- FastAPI
- Uvicorn (ASGI server)
- Pydantic (data validation)
- python-dotenv (environment variables)
